# Image Moderation System

## Description
A deep learning–based image moderation system that classifies images into multiple content categories using a CNN model. The system includes model training, evaluation, and a Flask-based API for real-time inference.

## Features
- Multi-class NSFW image classification
- Pretrained CNN-based model
- REST API using Flask for image prediction

## Files
- `nsfw_classification.py` – Model training and evaluation
- `flask_app.py` – Flask API for deployment
- `nsfw_5_classes_final.h5` – Trained model

## Tech Stack
- Python
- TensorFlow / Keras
- Flask
